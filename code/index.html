Étape : créer la structure HTML de base

Code HTML de base : ici nous allons simplement définir un menu, grâce à la balise nav, qui contiendra une liste non ordonnée (ul), qui contiendra des titres (non cliquables). Pour modéliser les sous-menus, nous allons utiliser aussi des listes, qui contiendront des liens.

Solution : 
		 <nav>
    <ul class="dropdownmenu">
        <li><a href="#">Menu 1</a></li>
        <li><a href="#">Menu 2</a>

Étape : affichage horizontal

Nous allons maintenant faire en sorte que les titres s'alignent horizontalement. Plusieurs méthodes sont possibles ici. Il est en effet possible d'utiliser par exemple la propriété CSS display: inline-block;. Néanmoins, nous allons préférer une gestion automatique de la taille des élements. Pour cela, il est possible d'utiliser flex ou grid. Nous allons utiliser grid. Utilisez grid-template-columns pour afficher 4 colonnes de tailles identiques. Profitez-en pour:

  • supprimer les puces des listes des nav ul
  • supprimer toutes les marges des nav ul
  • center le texte de tous les élements dans le menu déroulant dropdownmenu
  • transformez les élements liens a en block (simplifie la gestion de sa boîte.

Rendu visuel:
Solution: 
      .dropdownmenu {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    text-align: center;
}

nav ul {
    list-style-type: none;
    margin: 0px;
    padding: 0px;
}

  nav ul ul {
    border: 1 solid black;
}

Étape : Faire disparaître et apparaître les menus

Pour faire disparaître des éléments HTML, il suffit de donner la valeur none à la propriété display. Pour le faire réapparaître, utiliser la pseudo-class :hover pour modifier ; la valeur de display à block.- Attention à bien choisir votre sélecteur CSS :

  • vous souhaitez que l'ul inférieur soit non visible;
  • quand votre souris passe sur un li, le ul inférieur doit apparaître.

Rendu visuel:
Solution: 
nav ul ul {
    display: none;
}
nav li:hover ul {
    display: block;
}
Vous pouvez constater que les éléments de la page se déplacent quand vous passez sur les élements. Pour éviter cela ajoutez ceci à votre CSS: 
nav ul ul {
    position: absolute; /* Sort le sous-menu du flux normal */
    top: 100%; /* Positionne le sous-menu en dessous de l'élément parent */
    left: 0;
    z-index: 1000; /* Assure que le sous-menu est devant le reste */
}
